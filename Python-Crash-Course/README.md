## 📂 Repository Structure & Key Learnings

Here is a breakdown of the completed modules and the concepts mastered in each:

### `chapter_1_variables/`
- Variables and naming conventions.
- Working with Strings (methods like `title()`, `upper()`, f-strings).
- Numbers (Integers, Floats) and constants.
- Writing clean comments.

### `chapter_2_lists/`
- Creating and accessing elements in a List.
- Modifying, adding (`append()`, `insert()`), and removing (`pop()`, `remove()`) elements.
- Sorting and organizing Lists (`sort()`, `sorted()`, `reverse()`).

### `chapter_3_loops/`
- Iterating over lists using `for` loops.
- Avoiding indentation errors.
- Numerical lists using the `range()` function.
- Simple statistics with lists (`min`, `max`, `sum`).
- List Comprehensions for concise code.

### `chapter_4_if_statement/`
- Conditional tests (`==`, `!=`, `>`, `<`, `>=`).
- Checking multiple conditions (`and`, `or`).
- `if-elif-else` chains.
- Using `if` statements with Lists (checking for empty lists, multiple lists).

### `chapter_5_dictionaries/`
- Working with Key-Value pairs.
- Adding, modifying, and removing dictionary items.
- Iterating through dictionaries (`.keys()`, `.values()`, `.items()`).
- **Nesting:** 
  - A List of Dictionaries.
  - A List in a Dictionary.
  - A Dictionary in a Dictionary.
